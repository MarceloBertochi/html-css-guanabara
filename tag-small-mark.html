Podemos usar a tag big para aumentar uma letra ou um texo mas perceba que no editor ele ficou em vermelho, isso porque essa tag já esta absoleta (não se usa mais em html5).

Agora a tag small podemos usar tranquilamente para deixar a letra ou texto menor do resto do texto que queremos.

podemos usar o mark para deixar um texto marcado. essa marcação sempre será em amarelo, como um marcador de texo, caso queira mudar a cor, podemos fazer isso com o CSS.
